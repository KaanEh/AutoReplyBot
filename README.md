# 🤖 AutoReplyBot

**AutoReplyBot** is a simple BetterDiscord plugin that automatically replies to direct messages when you're away or unavailable — like an answering machine for Discord.

---

## ✨ Features

* 💬 Auto-replies to incoming DMs
* ⚡ Instant response
* 🧠 Simple and lightweight
* 🔧 Easy to customize message

## 📥 Installation

1. Download the plugin from this repository

   * Click **Code → Download ZIP**
   * Or download the file directly

2. Extract the ZIP (if needed)

3. Move the plugin file:

```
AutoReplyBot.plugin.js
```

into your BetterDiscord plugins folder:

```
C:\Users\<YourUsername>\AppData\Roaming\BetterDiscord\plugins
```

👉 Replace `<YourUsername>` with your Windows username.

---

### 💡 Alternative (Quick Access)

You can also open the folder directly:

1. Press `Win + R`
2. Paste this:

```
%appdata%\BetterDiscord\plugins
```

3. Press Enter
4. Drop the plugin file there

---

## ⚙️ Usage

* Once enabled, the bot will automatically respond to incoming direct messages
* You can edit the reply message inside the plugin file if needed

Example reply:

```
Hey ${user}! I'm currently not available and will get back to you later.
```

---

## 🛠️ Customization

To change the auto-reply message:

1. Open `AutoReplyBot.plugin.js`
2. Find the message string
3. Edit it to your liking
